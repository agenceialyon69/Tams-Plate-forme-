# TAMS — Stack Free-First

> Philosophie : jamais une dépendance payante si une alternative gratuite solide existe.

## Stack actuelle (gratuite)

| Besoin | Solution | Statut |
|---|---|---|
| Frontend | React + Vite + Tailwind | Gratuit |
| Backend | Express 5 + Drizzle | Gratuit |
| DB | PostgreSQL (Railway free tier) | Gratuit |
| AI | gemini-2.5-flash via Replit AI Gateway | Gratuit sur Replit |
| Codegen | Orval (OpenAPI) | Gratuit |
| Build | esbuild + Vite | Gratuit |
| Déploiement | Railway + Nixpacks | Free tier |

## Alternatives gratuites à privilégier

|---|---|

## Règle

Toujours évaluer : utilité quotidienne, maintenance, complexité, ROI, coût, durabilité.
Si une solution gratuite existe, l'utiliser en priorité.
