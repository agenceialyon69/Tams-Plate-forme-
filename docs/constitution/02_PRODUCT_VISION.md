# 02 — Vision Produit

## Modules prioritaires

1. **Chief of Staff** (Accueil) — briefing intelligent, priorités, risques, recommandations
2. **Chat OS** — cœur du système, 90% des actions démarrables depuis le chat
3. **Memory Graph** — mémoire relationnelle queryable, actionnable, avec edges typées
4. **Decision OS** — décisions structurées, score analytique, lien vers tâches
5. **Work OS** — tâches, projets, CRM léger, contacts, agenda
6. **Studio** — assets créatifs, prompts, templates, scripts
7. **Système** — audit, recovery, export, configuration
8. **Mobile premium** — expérience native, fluide, haut de gamme

## Non-objectifs

- Pas un dashboard de métriques
- Pas un CRM générique
- Pas un clone Notion / éditeur de documents
- Pas un SaaS multi-tenant avec plans payants
- Pas un clone ChatGPT avec chat isolé
- Pas un admin dashboard
- Pas de feature theater
- Pas de modules sans valeur quotidienne prouvée

## Philosophie free-first

Jamais une dépendance payante obligatoire.  
Prioritaire : Ollama local, Qwen, DeepSeek, OpenRouter gratuit, gemini-2.5-flash via Replit AI Gateway.

## Règle de l'écran quotidien

Avant chaque écran : « Est-ce que je l'ouvrirais tous les jours ? »  
Si non : fusionner, simplifier ou supprimer.
